RUN COMMAND:
python sokoban.py -l levelx.txt -m method_name

With:
x is the number level in file microcomos
method_name is "astar" or "dfs" or "bfs"

After entering the running command, please wait until it complete

Then, the game UI will appear. By typing space key, it will automatically move following the result step set.

Notice:
When you create the sokoban map, please notify the following about the character
'#': wall
' ': free space
'B': box
'X': box on goal
'.': goal
'W': player on goal
